# AGENTS.md

Правила для AI при работе с этим репозиторием:

1. Не ломать существующие working flows без необходимости.
2. Перед изменениями сначала кратко описывать план.
3. Все новые интеграции класть в shared/webhooks или shared/utils, если они переиспользуемые.
4. Не хранить секреты в коде.
5. Все настройки через .env.local или секреты LiveKit Cloud.
6. Для каждого нового webhook делать отдельный модуль.
7. Не переименовывать файлы и папки без явной причины.
8. После изменений проверять imports и точки входа.
9. Код должен быть простым и понятным для не-программиста.
