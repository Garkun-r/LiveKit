# LiveKit

Монорепозиторий для LiveKit-агентов.

Структура:
- agents/ — отдельные агенты
- shared/ — общие модули, веб-хуки, утилиты, промпты
- docs/ — документация
- .agents/ — инструкции и skills для AI

Первый агент:
- agents/main-bot
